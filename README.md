# Station Support App (Prototype)

This is a minimal Next.js prototype that shows:
- sample train display images
- board / do not board guidance
- platform number
- get-off station
- warning notice

## Run locally

```bash
npm install
npm run dev
```

Then open:
`http://localhost:3000`
